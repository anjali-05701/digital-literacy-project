Project Overview

This repository contains my Digital Literacy Project, developed as part of the CSE0001 – Digital Literacy course at VIT Bhopal University.

The goal of this project is to understand the importance of digital awareness, professional online presence, cyber safety, and effective digital communication in today's technology-driven world.

Through this project, I explored multiple digital platforms, created professional profiles, practiced coding basics, learned proper email etiquette, and studied cybercrime awareness. This project helped me enhance both my technical knowledge and responsible digital behavior.

Project Objectives

The key objectives of this project were to:

Understand the concept of Digital Literacy
Develop a professional digital portfolio
Explore coding practice platforms
Learn professional email communication
Understand cybercrime and prevention techniques
Build awareness about responsible internet usage

Tasks Completed
Task 1 – Digital Literacy Infographic

In this task, I created an awareness infographic explaining:

What digital literacy means
Safe internet usage
Useful digital tools for students
Responsible online behavior

The infographic was designed using Canva to present information in a visual and engaging format.

Task 2 – Digital Portfolio Creation

For this task, I created professional profiles on major learning and networking platforms including:

GitHub – to host and manage coding projects
LinkedIn – to build a professional network
Kaggle – to explore data science learning resources

This task helped me understand how maintaining an online professional presence can support learning and career development.

Task 3 – Coding & Collaboration Platforms

In this task, I explored coding and collaboration platforms by:

Solving a beginner coding problem on HackerRank
Creating a Digital Literacy quiz using Google Forms

This activity helped improve my problem-solving skills, coding exposure, and understanding of online collaboration tools.

Task 4 – Professional Email Writing

This task focused on professional digital communication.

Activities completed:

Drafted formal email samples
Created a Social Media Etiquette Checklist

Through this task, I learned how to communicate professionally in academic and workplace environments and avoid common mistakes in digital communication.

Task 5 – Cybercrime Case Study

For this task, I studied the Phishing Cybercrime case.

I researched:

How phishing attacks occur
Real examples of phishing scams
Methods to identify suspicious messages
Prevention strategies for students

I also created a Cyber Safety Checklist to spread awareness about protecting personal data online.
