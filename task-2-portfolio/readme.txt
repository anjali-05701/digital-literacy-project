# Task 2 – Student Digital Portfolio

## Overview
As part of the Digital Literacy project, I created professional profiles on important digital platforms that are widely used by students, developers, and researchers. These platforms help in building a strong digital presence, showcasing skills, and connecting with the global academic and professional community.

The platforms I selected are GitHub, LinkedIn, and Kaggle. Each platform serves a different purpose and together they form the foundation of a student's digital portfolio.

---

## Platforms Created

### 1. GitHub
GitHub is a platform used for version control and collaborative software development. It allows developers to store, manage, and share code repositories.

I created a GitHub profile and also added a Profile README which introduces me, my academic background, and my interests in technology. In the future, I plan to use GitHub to upload my programming projects, coursework, and coding practice so that it acts as a public portfolio of my technical work.

Screenshot: `github_profile.png`

---

### 2. LinkedIn
LinkedIn is a professional networking platform used by students, professionals, and recruiters. It helps individuals present their educational background, achievements, and career interests.

I updated my LinkedIn profile with my current education at VIT Bhopal University and basic academic details. Over the next four years, I plan to use LinkedIn to connect with professionals, follow technology companies, share my learning progress, and explore internship opportunities.

Screenshot: `linkedin_profile.png`

---

### 3. Kaggle
Kaggle is a popular platform for data science and machine learning enthusiasts. It provides datasets, coding notebooks, and competitions where users can practice data analysis and machine learning.

I created a Kaggle profile to begin exploring the field of data science. In the coming years, I plan to use Kaggle to learn Python for data analysis, work on datasets, and participate in beginner-level competitions to improve my analytical and programming skills.

Screenshot: `kaggle_profile.png`

---

## Reflection
Creating profiles on these platforms helped me understand the importance of maintaining a professional digital presence. These platforms will help me showcase my learning journey, collaborate with others, and build a strong portfolio throughout my undergraduate studies.
